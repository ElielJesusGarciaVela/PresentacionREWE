## Introducción a la empresa:

Rewe Digital Spain es una empresa dentro de REWE Group, es la rama de IT o rama técnica de REWE Group.

El trabajo realizado varia mucho, desde Java y Kotlin hasta ABAP. Y se especializa en campos como Financas, Recursos Humanos, Producción, etc...

Nosotros hemos estado trabajando con ABAP, que es el lenguaje de SAP, la empresa más grande de toda Europa.

## <br>Tareas desempeñadas con temporalización por semanas:


Semana 1

- Realizacion de un curso de ABAP para principiantes para poder empezar a programar.
- Creacion de pequeños trozos de codigo con ayuda del curso para tener un ejemplo practico.

Semana 2

- Terminamos el curso de ABAP
- Realizacion de las primeras tareas de confluence, como una print hello world, una clase que suma, etc.

Semana 3

- Creacion de nuestro primer proyecto, una calculadora basica con interfaces y su clase.

Semana 4

- Creacion de un ABAP Report/Program para que el usuario pueda interactuar de forma gráfica con el programa.
calculadora con abap report/program

Semana 5

- Creacion de tablas y sus respectivos elementos necesarios dentro de ABAP para el futuro proyecto de ticket machine 

Semana 6

- Union de tablas entre si mediante relaciones, y creacion de clases para que los usuarios puedan rellenar las tablas con información

Semana 7

- Creacion de una clase que rellene las tablas con datos aleatorios dentro de un rango y mostrarla

Semana 8

- Creacion de unit tests para la calculadora y otras tareas previas

Semana 9

- Aprender como representar informacion en forma de tablas usando abap alv salv
- Creacion de una clase que represente las tablas de ticket machine usando salv
